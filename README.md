# Telegram Translator Bot

Внутренний Telegram-бот для перевода текста и документов для сотрудников компании.

Бот умеет:

- переводить текст из сообщений
- переводить файлы `.txt`, `.docx`, `.pdf`
- распознавать текст на изображениях
- распознавать текст на сканированных PDF (OCR)
- переводить технические требования с чертежей
- хранить доступ пользователей в PostgreSQL
- логировать переводы

---

# Возможности

### Поддерживаемые файлы

Бот может переводить:

- `.txt`
- `.docx`
- `.pdf`
- `.png`
- `.jpg`
- `.jpeg`
- `.tif`
- `.tiff`
- `.bmp`

Для сканированных документов используется OCR (Tesseract).

---

# Режимы перевода

Бот поддерживает два режима.

### Обычный режим

Подходит для:

- документов
- писем
- инструкций
- текстовых файлов

### Режим чертежа

Используется для инженерных документов.

Бот:

- переводит текстовые требования
- не изменяет размеры
- не изменяет номера позиций
- не изменяет обозначения стандартов
- не изменяет марки материалов

---

# Использование

1. Откройте бота в Telegram
2. Нажмите `/start`
3. Выберите направление перевода
4. Выберите режим
5. Отправьте текст или файл

Бот автоматически переведёт содержимое.

---

# Основные команды

| Команда | Описание |
|-------|-------|
| `/start` | запуск бота |
| `/help` | помощь |
| `/whoami` | показать Telegram user ID |
| `/ruen` | перевод русский → английский |
| `/enru` | перевод английский → русский |
| `/drawing` | режим чертежа |
| `/normal` | обычный режим |

# Администрирование

Команды администратора:

## Требования

- Go 1.22+
- PostgreSQL
- Tesseract OCR
- Poppler Utils

## Установка OCR

Ubuntu:


sudo apt update
sudo apt install poppler-utils tesseract-ocr tesseract-ocr-rus tesseract-ocr-eng

# OCR

Если PDF содержит текст — он извлекается напрямую.

Если PDF является сканом:

1. страницы конвертируются в изображения
2. текст распознаётся через Tesseract
3. распознанный текст переводится

---

# Логи переводов

Бот сохраняет в базе:

- пользователь
- направление перевода
- тип файла
- имя файла
- количество символов
- успешность перевода
- дата
