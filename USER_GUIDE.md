# Инструкция для сотрудников: Telegram-бот переводчик

## Что умеет бот

Бот умеет:

- переводить текст из сообщений
- переводить файлы `.txt`, `.docx`, `.pdf`
- распознавать текст на изображениях
- распознавать текст на сканированных PDF, TIFF и изображениях через OCR
- переводить технические требования с чертежей

## Как пользоваться ботом

### Шаг 1. Откройте бота
Найдите бота в Telegram и нажмите **Start**.

### Шаг 2. Узнайте свой Telegram ID
Отправьте команду:

```text
/whoami
```

Бот пришлёт ваш `user_id`.

### Шаг 3. Отправьте этот ID администратору
Например так:

```text
Мой Telegram ID: 5306182091
Прошу выдать доступ к боту.
```

### Шаг 4. Администратор выдаёт доступ
Администратор использует команду:

```text
/allow_id 5306182091
```

После этого бот станет доступен.

## Как пользоваться ботом

### 1. Нажмите `/start`
Бот покажет кнопки с настройками.

### 2. Выберите направление перевода

- **Рус → Англ**
- **Англ → Рус**

### 3. Выберите режим

- **Обычный режим** — для обычных документов, писем, инструкций, текста
- **Режим чертежа** — для технических требований, сканов, TIFF, PDF чертежей и инженерных документов

### 4. Отправьте текст или файл

Можно отправлять:

- текст сообщением
- `.txt`
- `.docx`
- `.pdf`
- `.png`
- `.jpg`
- `.jpeg`
- `.tif`
- `.tiff`
- `.bmp`

## Какой режим выбирать

### Обычный режим
Используйте для:

- писем
- инструкций
- текстовых PDF
- Word-документов
- обычных текстов

### Режим чертежа
Используйте для:

- сканированных PDF
- TIFF-файлов
- изображений с техтребованиями
- инженерных чертежей
- фотографий или сканов листов с требованиями

В этом режиме бот старается:

- извлечь только технические требования
- убрать OCR-мусор
- не менять размеры
- не менять коды и номера
- не менять стандарты и марки материалов

## Примеры сценариев

### Перевод обычного текста
1. Нажмите `/start`
2. Выберите **Рус → Англ**
3. Выберите **Обычный режим**
4. Отправьте текст

### Перевод PDF
1. Выберите направление
2. Выберите режим
3. Отправьте PDF

### Перевод чертежа или TIFF
1. Выберите направление
2. Выберите **Режим чертежа**
3. Отправьте `.tif`, `.tiff`, `.pdf` или изображение

## Полезные команды

```text
/start    — открыть бота и показать меню
/help     — краткая помощь
/status   — показать текущие настройки
/whoami   — показать ваш Telegram ID
```

## Что делать, если бот пишет «Нет доступа»

1. Отправьте `/whoami`
2. Скопируйте свой ID
3. Отправьте ID администратору
4. Дождитесь, пока администратор выдаст доступ

## Что делать, если перевод плохой

### Для чертежей и сканов
Проверьте, что выбран **Режим чертежа**.

### Для обычных документов
Проверьте, что выбран **Обычный режим**.

## Ограничения

- качество OCR зависит от качества скана
- сильно шумные чертежи могут распознаваться неидеально
- очень мелкий текст может читаться хуже
- если документ плохо читается человеком, бот тоже может распознать его с ошибками

## Кому писать по вопросам доступа

Обратитесь к администратору бота и отправьте свой `user_id`, который бот показывает по команде `/whoami`.
